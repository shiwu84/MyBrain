## 课程大纲

**注册系统以获取红帽支持**
使用您的红帽（Red Hat）帐户注册系统，以便获取红帽提供的支持服务和软件。

**通过命令行管理文件**
在命令行中复制、移动、创建、删除和组织文件。

**编辑文本文件**
通过命令行创建、查看和编辑文本文件。

**管理本地用户和组**
获取系统的超级用户访问权限；创建、管理和删除本地用户及组；并管理本地密码策略。

**控制文件访问权限**
设置文件的标准权限，并解读不同权限设置对安全性的影响。

**使用 RPM 安装和更新软件**
从红帽和 DNF 软件包存储库下载、安装、更新和管理软件包。

**使用 Flatpak 安装和更新应用程序**
使用 Flatpak 从红帽生态系统目录（Red Hat Ecosystem Catalog）安装、升级和使用桌面软件。

**访问可移动介质**
通过将可移动介质设备挂载到文件系统层次结构中的目录上，来访问其上的文件系统。

**监控和管理 Linux 进程**
调查、控制和终止在 Red Hat Enterprise Linux 系统上运行的进程。

**控制服务和守护进程**
控制和监控由 systemd 启动的系统服务和守护进程。

**管理网络配置**
配置 Red Hat Enterprise Linux 服务器上的网络接口和设置。

**调度系统任务**
调度（计划）必须定期运行的系统程序，以支持守护进程或操作系统功能。

**分析和存储日志**
定位并解读系统日志以进行故障排除，并确保日志事件的时间戳准确。

**使用 SELinux 管理安全性**
使用 SELinux 保护系统并管理安全性。

**管理基础存储**
通过命令行创建分区、文件系统和交换空间（swap）来管理存储设备。

**使用逻辑卷管理器（LVM）管理存储**
使用逻辑卷管理器 (LVM) 管理可包含文件系统和交换空间的逻辑卷。

**控制和故障排除引导过程**
管理系统的引导方式，以控制启动哪些服务，并排除和修复引导（Boot）时的问题。

**恢复超级用户访问权限**
当超级用户密码未知或被锁定时，获取系统的管理访问权限。

**管理网络安全**
使用系统防火墙控制对服务的网络连接，并利用 SELinux 控制网络服务可绑定的特定端口。

**访问网络附加存储**
访问使用网络文件系统 (NFS) 协议提供的网络附加存储（手动访问或使用自动挂载器）。

**综合复习**
练习在 *RHCSA 速成课程*（RHCSA Rapid Track Course*）中学到的技能。
