# 红帽认证系统管理员（RHCSA）考试

## 考试要点

RHCSA 考生应能在没有他人帮助的情况下完成下列任务。这些任务分为多个类别。

### 了解和使用基础工具

- 访问 shell 命令提示符，并使用正确的语法发出命令
- 使用输入输出重定向（>、>>、|、2> 等）
- 使用 grep 和正则表达式来分析文本
- 使用 SSH 访问远程系统
- 在多用户目标中登录和切换用户
- 使用 tar、gzip 和 bzip2 将文件归档、压缩、解除捆绑和解压缩
- 创建和编辑文本文件
- 创建、删除、复制和移动文件与目录
- 创建硬链接和软链接
- 列出、设置和更改标准 ugo/rwx 权限
- 查找、读取和使用系统文档，包括 /usr/share/doc 中的手册、信息和文件

### 管理软件

- 配置对 RPM 存储库的访问权限
- 安装和删除 RPM 软件包
- 配置对 Flatpak 存储库的访问权限
- 安装和删除 Flatpak 软件包

### 创建简单的 shell 脚本

- 在一定条件下执行代码（使用：if、test、[] 等）
- 使用循环结构（for 等）来处理文件、命令行输入
- 处理脚本输入（$1、$2 等）
- 在脚本中处理 shell 命令的输出

### 运维运行中的系统

- 正常启动、重启和关闭系统
- 手动让系统启动进入不同的目标
- 中断启动流程，获取系统访问权限
- 识别 CPU/内存占用量较大的进程，并停止相关进程
- 调整进程的调度安排
- 管理调优配置文件
- 查找和解读系统日志文件和日志信息
- 保留系统日志
- 启动、停止网络服务，以及查看网络服务的状态
- 在系统间安全地传输文件

### 配置本地存储

- 罗列、创建和删除 GPT 磁盘上的分区
- 创建和删除物理卷
- 将物理卷分配至卷组
- 创建和删除逻辑卷
- 配置系统，以便在启动时根据通用唯一 ID（UUID）或标签来挂载文件系统
- 添加新的分区和逻辑卷，在保留原有结构的情况下置换为某个系统

### 创建和配置文件系统

- 创建、挂载、卸载、使用 vfat、ext4 和 xfs 文件系统
- 使用 NFS 挂载和卸载网络文件系统
- 配置 autofs
- 扩展现有逻辑卷
- 诊断和修复文件权限问题

### 部署、配置和维护系统

- 使用 cron 和 systemd 计时器单元来调度任务
- 启动和停止服务，配置服务以在系统启动时自动启动
- 配置系统，以便自动启动进入某个特定目标
- 配置时间服务客户端
- 通过红帽内容分发网络、远程存储库或本地文件系统来安装和更新软件包。
- 修改系统引导加载程序

### 进行基本联网管理

- 配置 IPv4 和 IPv6 地址
- 配置主机名称解析
- 配置网络服务，以便在系统启动时自动启动
- 使用 firewalld 和 firewall-cmd 限制网络访问

### 管理用户和群组

- 创建、删除和修改本地用户帐户
- 为本地用户帐户更改密码和调整密码有效期
- 创建、删除和修改本地群组与群组成员资格
- 配置特权访问

### 管理安全防护

- 使用 firewall-cmd/firewalld 配置防火墙设置
- 管理默认文件权限
- 为 SSH 配置基于密钥的身份验证机制
- 为 SELinux 设置强制和允许模式
- 列出和识别 SELinux 文件与进程的上下文
- 还原默认文件上下文
- 管理 SELinux 端口标签
- 使用布尔值设置来修改系统 SELinux 设置

对于所有实际任务操作型红帽考试，在重新启动后，您的所有系统配置必须在没有人工干预的情况下仍然有效。

# 红帽系统管理一

## **课程大纲**

### **红帽企业 Linux 入门**

介绍开源、Linux、Linux 发行版和红帽企业 Linux，并明确相关定义。

### **访问命令行**

登录 Linux 系统并使用 shell 运行简单的命令。

### **通过本地文档获取帮助**

使用本地帮助系统和文档中的信息解决问题。

### **注册系统以获取红帽支持**

使用您的红帽帐户注册系统，以获得红帽提供的支持服务和软件。

### **利用 RHEL Lightspeed 获取 AI 辅助帮助**

利用红帽企业 Linux Lightspeed 获取 AI 驱动的指导和建议，以高效管理 RHEL 服务器。

### **浏览文件系统层次结构**

描述 Linux 文件系统层次结构中的文件组织，并使用相对路径或绝对路径按名称指定文件。

### **通过命令行管理文件**

使用命令行复制、移动、创建、删除和整理文件。

### **编辑文本文件**

使用命令行创建、查看和编辑文本文件。

### **重定向 shell 输入和输出**

使用 Bash 重定向命令的输出和输入。

### **管理本地用户和群组**

获取系统的超级用户访问权限；创建、管理和删除本地用户与群组，并管理本地密码策略。

### **控制文件访问权限**

设置文件的标准权限，并解释不同权限设置的安全效果。

### **使用 RPM 安装和更新软件**

从红帽和 DNF 软件包存储库下载、安装、更新和管理软件包。

### **使用 Flatpak 安装和更新应用**

使用 Flatpak 安装、升级和使用红帽生态系统目录中的桌面软件。

### **访问可移动介质**

通过将可移动介质设备挂载到文件系统层次结构中的目录来访问文件系统。

### **监控和管理 Linux 进程**

调查、控制和终止在红帽企业 Linux 系统上运行的进程。

### **控制服务和守护进程**

控制和监控 systemd 启动的系统服务和守护进程。

### **网络入门**

描述基本的 TCP/IP 网络概念，并调查服务器上当前的网络配置和功能。

### **管理网络配置**

配置红帽企业 Linux 服务器上的网络接口和设置。

### **配置和保护 SSH**

通过管理主机密钥并为用户实施基于密钥的身份验证来保护 SSH 通信。

### **综合复习**

练习在红帽系统管理一中学到的技能。

# 红帽系统管理二

## 课程大纲

### Shell 脚本编写和命令行

编写和运行简单的 shell 脚本，并使用 shell 脚本编写功能在 shell 提示符下高效运行命令。

### 在实际应用中使用正则表达式

使用正则表达式匹配文本，高效完成系统管理任务。

### 调度用户任务

以普通用户身份调度程序在未来运行，可在指定时间和日期运行，也可周期性运行。

### 调度系统任务

调度那些必须要定期运行的系统程序，以支持守护进程或操作系统功能。

### 分析和存储日志

查找和解读系统日志以用于故障排除，并确保日志事件的记录时间准确无误。

### 使用 SELinux 进行安全管理

使用 SELinux 保护系统和管理安全防护。

### 将文件存档

创建文件的压缩存档，以便备份并传输到其他系统。

### 传输文件

在系统之间安全地传输文件。

### 调优系统性能

通过设置调优配置集和调整特定进程的调度优先级来提高系统性能。

### 管理基本存储

通过从命令行创建分区、文件系统和交换空间来管理存储设备。

### 使用逻辑卷管理器管理存储

使用逻辑卷管理器（LVM）来管理可包含文件系统和交换空间的逻辑卷。

### 控制启动流程并进行故障排除

管理系统的启动方式，以控制启动哪些服务，并对启动时问题进行故障排除和修复。

### 恢复超级用户访问权限

当超级用户密码未知或被锁定时，获取对系统的管理访问权限。

### 管理网络安全

使用系统防火墙控制与服务的网络连接，并使用 SELinux 控制那些可绑定到特定端口的网络服务。

### 访问网络附加存储

手动或使用自动挂载器访问网络文件系统（NFS）协议提供的网络附加存储。

### 安装红帽企业 Linux

以软件包形式安装红帽企业 Linux，包括通过交互方式或使用 Kickstart 进行安装。

### 使用 Podman 管理容器

使用最新版容器管理工具管理容器和容器镜像。

### 使用基于镜像的红帽企业 Linux

创建、使用、安装和升级使用基于镜像的安装管理的容器和服务器。

### 综合复习

练习您在_红帽系统管理二_中学到的技能。
